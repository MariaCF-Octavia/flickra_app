The Technical Solution

Personalized brand training + real-time mobile guidance

How The System Works

1. Brand Training (30-minute setup)

Businesses upload 50-100 product images from multiple angles and lighting conditions. YOLOv8 training on AWS (ml.g5.2xlarge, ~$3 per customer) creates a personalized model that learns:

  • Optimal angles and lighting for their specific product
  • Brand-specific quality standards
  • Logo placement and composition requirements

Not generic object detection—personalized quality standards per business.

2. Real-Time Mobile Guidance (Flutter app, 30+ FPS)

Camera interface with live feedback powered by the trained model:

  • Quality score (0-100): Capture disabled until achieving 85+ score
  • Real-time coaching: "Move left," "Increase lighting," "Adjust angle"
  • Background preview: View final commercial environment before shooting

3. Multi-API Orchestration

Once optimal shot captured:

  • Runway → Background replacement (brand-trained)
  • VEO3 → Video generation
  • CLAID → Multi-scene creation + enhancement
  • RealESRGAN → Upscaling
  • Output: 5 complete commercials from one optimized shot

Competitive moat: Every customer training session improves the guidance system through automated retraining. Network effects from usage, not just scale.

Technical Architecture Detail

System components and integration approach

Brand Training Pipeline

Input: Customer uploads 50-100 product images

Processing:

  • YOLOv8 training on AWS ml.g5.2xlarge (~$3/customer, 1-2 hours)
  • Model storage on S3 (~$0.50/month per customer)
  • Automated retraining triggers improve model accuracy over time

Output: Personalized quality scoring model understanding specific brand standards

Mobile Real-Time Inference

Challenge: Dedicated endpoints cost $876/month per customer (not viable at $149-599 pricing)

Solution: Lambda-based serverless inference

  • Model cached in Lambda container (~$0.003 per request)
  • Smart caching maintains "warm" state during active sessions
  • Flutter app optimized for 30+ FPS with intelligent frame sampling

Note: Achieves "near real-time" (2-3 second latency) rather than instant feedback. Acceptable for guided production workflow.

Multi-Scene Video Generation

User defines commercial structure: "Product on table → Application in bathroom → Friends in car → Call-to-action"

Processing chain:

  • Runway: Background replacement for each scene (brand-trained model)
  • VEO3: Image-to-video conversion
  • CLAID: Multi-scene assembly + enhancement
  • RealESRGAN: Upscaling for quality optimization

Per-generation cost: ~$2-4 depending on complexity and scene count

Infrastructure Stack

  • Mobile development: Flutter (iOS + Android from unified codebase)
  • Computer vision: Ultralytics HUB + YOLOv8
  • Compute: AWS Lambda (inference) + SageMaker (training)
  • Storage & delivery: S3 (models + videos) + CloudFront (CDN)
  • Monitoring: CloudWatch + custom analytics dashboard
